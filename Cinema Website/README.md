## Running the Application


### To Run Client-App
#### Run express
1) `cd main_website`
2) `cd express`
3) `npm start`
#### Run react
1) `cd main_website`
2) `cd react`
3) `npm start`

### To Run Admin-Dashboard
#### Run express
1) `cd admin-dashboard`
2) `cd express`
3) `npm start`

#### Run react
1) `cd admin-dashboard`
2) `cd admin`
3) `npm run start-pc` for Windows or
4) `npm run start-mac` for Mac
